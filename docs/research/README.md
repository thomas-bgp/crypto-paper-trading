# Research em Andamento

Diretorio para pesquisas ativas. O prompt de varredura de artigos sobre funding rate deposita achados aqui.

## Como integrar novos achados

1. Novos papers/artigos devem ser adicionados a `docs/knowledge/literatura-academica.md` na secao tematica correta
2. Insights operacionais (mecanica de exchange, edge pratico) vao em `docs/knowledge/funding-rate-practitioners.md`
3. Se uma pesquisa leva a uma nova estrategia candidata, criar `docs/strategies/[nome].md`
4. Decisoes do comite vao em `docs/decisions/YYYY-MM-DD-titulo.md`
